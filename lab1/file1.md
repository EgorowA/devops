WSL - це прошарок між windows та linux, який створений для сумісного використання windows та linux в windows. WSL дозволяє втсановити дистрибутив linux в windows
та використовувати додатки, службові програми та командну строку Bash.
Інсталяція:
1. Запустити powershell під адміном
2. Інсталювати дистрибутив використовуючи комманду wsl -install -d <name>
